三、程式介紹
(1)操作步驟
1.進入歡迎介面
2.輸入顧客性別( m or f )
3.輸入是否指定設計師 ( y or n )
4.輸入是否需要剪髮服務 ( y or n )
5.輸入選擇是否只剪瀏海(女性)
6.是否需要染髮服務 ( y or n )
7.如有需要染髮，選擇染劑 
( 1為一般染劑，2為天然護髮染)
8.輸入是否需要護髮服務 ( y or n )
9.輸入是否需要燙髮服務 ( y or n )
10.如有需要燙髮，選擇燙髮項目
( 1為髮根澎澎燙，2為局部燙，3為哥德式閃亮)
11.輸入是否需要洗髮服務 ( y or n )
12.輸入是否繼續預約下一位顧客 ( y or n )
(如有需要，返回步驟2.)
13.系統顯示相關交易明細。

(2)附加功能
1.新增抽獎功能
如預約人數共大於等於5人，可獲得抽獎機會(95折、7折、50元折扣)

2.新增錯誤輸入提示功能
如輸入的字串並非系統要求，系統將提醒"錯誤"並提醒使用者重新輸入

3.新增交易明細功能
在預約完成後，系統會輸出所有使用者的選擇、預約時間(預約當下時間)等相關資訊
